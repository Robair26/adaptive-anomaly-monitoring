# Week 3 – Literature Sources for Anomaly Detection

## 1. Numenta Anomaly Benchmark (Dataset)

Lavin, A., & Ahmad, S. (2015). Evaluating real-time anomaly detection algorithms – The Numenta anomaly benchmark. IEEE International Conference on Machine Learning and Applications (ICMLA).

This paper introduces the Numenta Anomaly Benchmark (NAB), a widely used dataset for evaluating anomaly detection algorithms on streaming time-series data.

---

## 2. Isolation Forest

Liu, F. T., Ting, K. M., & Zhou, Z. H. (2008). Isolation forest. Proceedings of the 2008 IEEE International Conference on Data Mining.

Isolation Forest is an unsupervised anomaly detection algorithm that isolates anomalies using random partitioning in decision trees.

---

## 3. LSTM for Anomaly Detection

Malhotra, P., Vig, L., Shroff, G., & Agarwal, P. (2015). Long short term memory networks for anomaly detection in time series. European Symposium on Artificial Neural Networks.

This work demonstrates how LSTM networks can model temporal behavior in sequential data and detect anomalies based on reconstruction error.

---

## 4. Time Series Anomaly Detection Survey

Chalapathy, R., & Chawla, S. (2019). Deep learning for anomaly detection: A survey.

This paper reviews modern anomaly detection approaches including statistical methods, machine learning techniques, and deep learning models.

---

## How These Sources Relate to the Project

This capstone project uses:

• NAB dataset for anomaly detection evaluation  
• Isolation Forest as the classical machine learning method  
• LSTM Autoencoder as the deep learning model  

These sources provide the academic foundation for the methods implemented in the project.
