# Adaptive Anomaly Monitoring

Adaptive System Behavior Monitoring & Anomaly Detection using ML baselines and LSTM Autoencoders.

## Objectives
- Learn normal system behavior from time-series telemetry
- Detect anomalous events with an anomaly score + thresholding
- Compare statistical baselines vs classical ML vs deep learning
- Produce interpretable diagnostics and strong evaluation visuals

## Dataset
Numenta Anomaly Benchmark (NAB)

## Project Structure
- `data/` - raw + processed data
- `notebooks/` - EDA, baselines, deep model, evaluation
- `src/` - reusable pipeline code (preprocess, train, evaluate, detect)
- `figures/` - plots used in report/presentation
- `reports/` - exported notebook PDFs and report assets
